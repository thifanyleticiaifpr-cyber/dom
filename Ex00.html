DOM e Formulários

DOM

DOM - Data Object Model é uma API(Applicationb programming Interface) que visa simplificar a forma de acessaer a estrututra de um documento HTML. Facilitando criar, modificar, adicionar e manipular elementos HTML.

Formulários

Formulários são elementos HTML que possibilita a interação do usuário com a página web. Como exemplo temos caixas de texto, botões, caixas de seleção dentre outros. As tags form, label, input são elementos de formulários.

Exemplo

Construa uma interface para que sejam digitados dois números e somdos.
